Modules

Username

  • key prefix: username:
  • /usernames -> returns bnsd/x/username/username.Token by id(8 bytes) or empty result
  • /usernames?prefix -> returns one or more Token that begin with that prefix
  • Message paths
    • username/register_token -> bnsd/x/username.RegisterTokenMsg
    • username/transfer_token -> bnsd/x/username.TransferTokenMsg
    • username/change_token_targets -> bnsd/x/username.ChangeTokenTargetMsg

Batch

  • Message paths
    • batch/execute_batch -> bnsd/ExecuteBatchMsg, bnsd/ExecuteProposalBatchMsg

Migration

  • Message paths
    • migration/upgrade_schema -> migration.UpdateSchemaMsg

Cash

  • key prefix: cash:
  • /wallets -> returns x/cash.Set by id (8 bytes) or empty result
  • /wallets?prefix -> returns one or more Set that begin with that prefix
  • Message paths
    • cash/send -> x/cash.SendMsg
    • update_configuration -> x/cash.UpdateConfigurationMsg

  • Message paths
    • multisig/create -> x/multisig.CreateMsg
    • multisig/update -> x/multisig.UpdateMsg

Atomic swaps

  • key prefix: swap:
  • /aswaps -> returns x/aswap.Swap by id (8 bytes) or empty result
  • /aswaps/source -> returns x/aswap.Swap by source(20 bytes address) as secondary index or empty result
  • /aswaps/destination -> returns x/aswap.Swap by destination(20 bytes address) as secondary index or empty result
  • /aswaps/preimage_hash -> returns x/aswap.Swap by preimage_hash(32 bytes sha256 hash) as secondary index or empty result
  • /aswaps?prefix -> returns one or more Swap that begin with that prefix
  • Message paths
    • aswap/create -> x/aswap.CreateMsg
    • aswap/release -> x/aswap.ReleaseMsg
    • aswap/return -> x/aswap.ReturnMsg

Escrow

  • key prefix: escrow:
  • /escrows -> returns x/escrow.Escrow by id(8 bytes) or empty result
  • /escrows/source -> returns x/escrow.Escrow by source(20 bytes address) as secondary index or empty result
  • /escrows/destination -> returns x/escrow.Escrow by destination(20 bytes address) as secondary index or empty result
  • /escrows/arbiter -> returns x/escrow.Escrow by arbiter (20 bytes address) as secondary index
  • /escrows?prefix -> returns one or more Escrow that begin with that prefix
  • Message paths
    • escrow/create -> x/escrow.CreateMsg
  • escrow/release -> x/escrow.ReleaseMsg
  • escrow/return -> x/escrow.ReturnMsg
  • escrow/UpdateParties -> x/escrow.UpdatePartiesMsg

Governance

Electorate

  • key prefix: electorate:
  • /electorate -> returns x/gov.Electorate by id(8 bytes) or empty result
  • /electorate?prefix -> returns one or more Electorate that begin with that prefix
  • Message paths
    • gov/update_electorate -> x/gov.UpdateElectorateMsg

Elector

  • Elector is a multikey index of electorate
  • key prefix: elector:
  • /electorate/elector -> returns x/gov.Elector by elector(20 bytes address) or empty result
  • /electorate/elector?prefix -> returns one or more Elector that begin with that prefix

Vote

  • key prefix: vote:
  • /vote -> returns x/gov.Vote by id(8 bytes) or empty result
  • /vote?prefix -> returns one or more Vote that begin with that prefix
  • /vote/proposal -> returns x/gov.Vote by id(8 bytes) or empty result
  • /vote/proposal?prefix -> returns one or more Vote that begin with that prefix -- /vote/elector -> returns x/gov.Vote by id(8 bytes) or empty result
  • /vote/elector?prefix -> returns one or more Vote that begin with that prefix
  • Message paths
    • gov/vote -> x/gov.VoteMsg
    • gov/tally -> x/gov.TallyMsg
